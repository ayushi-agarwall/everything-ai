## General Standards

### Quality Over Quantity
- We curate, not just collect
- Resources should provide genuine value
- Avoid duplicates or near-duplicates
- Focus on evergreen, high-quality content

### Formatting Rules

**Consistency is crucial:**
- Use exact table formatting from existing entries
- Maintain alphabetical order within subsections when appropriate
- Keep descriptions concise and informative
- Use proper markdown linking: `[Display Text](URL)`

**Date formats:**
- Use YYYY for years
- Use "Varies" only when truly variable
- Use "~X hours/weeks/months" for durations

**Pricing:**
- "Free" for completely free resources
- "Free (Audit) / $XX" for courses with paid certificates
- Specific amounts when known
- "Paid" only as last resort when amount varies

### Link Quality
- Always use HTTPS when available
- Link to official sources
- Avoid URL shorteners
- Test links before submitting

Before submitting, ensure:
- [ ] Resource is added to correct category
- [ ] All required fields are filled

- [ ] No duplicate entries exist
- [ ] Information is accurate and up-to-date
- [ ] PR description explains the value of this resource

## Review Process

1. **Submission**: Open a PR with your addition
2. **Review**: Maintainers will review for quality, accuracy, and formatting
3. **Feedback**: You may receive requests for changes
4. **Merge**: Once approved, your contribution will be merged

**Timeline**: Most PRs are reviewed within 1 week.

## Need Help?

- **Unsure about formatting?** Copy an existing entry and modify it
- **Not sure which category?** Open an issue to discuss
- **Found an error?** Open an issue or submit a fix
- **Have questions?** Open an issue with the `question` label

## Recognition
